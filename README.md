# LIPA

An ELM inspired, functional, vanilla Javascript framework.

Dependencies:
- express

devDependencies:
- virtual-dom
- watchify
- terser
- nodemon

# Install

```bash
# 1. Install dependencies.
npm i

# 2. Start development
npm run dev

# 3. Start the server to deliver the build
nodemon server.js

# 4. Build app
npm run build
```
